## 1. Introduction
The UPI Risk Intelligence System is an AI-based application designed to detect fraudulent UPI transactions using machine learning and real-time analytics.

## 2. System Requirements
- Python 3.9 or above
- Streamlit
- Internet Browser
- Windows / Linux OS

## 3. How to Run
1. Install dependencies  
2. Run the Streamlit application  
3. Login and analyze transactions

## 4. Features
- Fraud prediction using machine learning
- Transaction risk scoring
- Security scan and analytics dashboard
- Visual fraud network analysis

## 5. Conclusion
This system helps identify and prevent UPI fraud efficiently using AI-based techniques.
